# Análisis de requisitos

## Objetivo del cliente

La empresa necesita una infraestructura web segura y escalable para alojar:

* Página web corporativa
* Sistema de gestión interna

## Requisitos funcionales

* Servidor Apache con PHP
* Base de datos MariaDB
* Acceso remoto mediante SSH
* Firewall configurado con UFW
* Monitorización del sistema
* Sistema de copias de seguridad automáticas

## Requisitos no funcionales

* Seguridad básica endurecida
* Disponibilidad del servicio
* Facilidad de mantenimiento
* Documentación técnica clara

## Usuarios implicados

| Usuario       | Función                   |
| ------------- | ------------------------- |
| Administrador | Gestión del servidor      |
| Empleados     | Uso de aplicación interna |
| Clientes      | Acceso a la web           |
